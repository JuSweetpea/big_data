Для выгрузки (парсинга) данных с api.hh.ru используется код на питоне в программной реализации PyCharm.

Необходимо запустить "Код_парсинг_hh_ru.py"

Далее будет сохранён файл в формате json на используемой машине в указанной директории.

Запускаем программное обеспечение ELK.

Загружаем полученный json файл в elasticksearch.

Так как в данном исследование работаем без анализатора текста, то используем дополнительно python для анализа полученных данных.

Используемый код тоже запускаем "Анализ_данных.ipynb"
